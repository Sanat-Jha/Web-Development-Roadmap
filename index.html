Your Learning Journey

Day 1

How the Web Works & Setting Up

Learn what a website is, understand frontend vs backend, and set up your coding environment.

Days 2-3

HTML & CSS Mastery

Structure web pages with HTML and style them with CSS. Focus on understanding, not memorizing.

Days 5-10

JavaScript Fundamentals

Learn programming basics, DOM manipulation, and interactive web development.

Days 10-15

Build Your First Project

Create a creative project from scratch - to-do list, quiz app, or weather dashboard.
